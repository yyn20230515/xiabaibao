虾虾写作助手是给妈妈专用的 AI 写作小帮手。妈妈可以在这里和虾虾聊天，让虾虾帮写小说情节、续写/扩写段落、查资料等。

## 🖥️ 页面布局

### 三栏设计

```
┌──────────────┬──────────────────────┬──────────────┐
│  📋 对话列表  │      💬 聊天区        │  📁 文件管理  │
│              │                      │              │
│  对话1  ←选中 │  阿姨：帮我写个开头~   │  📂 全部文件  │
│  对话2       │                      │  📁 玄幻     │
│  对话3       │  虾虾：好嘞阿姨！...  │  📁 修仙     │
│  对话4       │                      │              │
│              │  [输入框...] [发送]   │  [📤 上传]   │
└──────────────┴──────────────────────┴──────────────┘
```

- **左栏**：对话列表，点选切换对话，可新建/删除
- **中栏**：聊天区，和虾虾对话的地方
- **右栏**：文件管理，上传小说文件后虾虾可以续写/扩写

左右栏都可以折叠，让聊天区全屏～

## 💬 如何使用

### 聊天
1. 在底部输入框打字，按回车或点「发送」
2. 虾虾会先回「收到啦！虾虾正在写～✍️」
3. 等一会儿收到虾虾的完整回复

### 文件管理
- **上传文件**：点右栏「📤 上传」，选 `.txt` 或 `.md` 文件
- **预览文件**：点文件列表里的文件名
- **续写/扩写**：预览文件后点「✍️ 续写」或「🔍 扩写」，虾虾会帮你接着写
- **下载**：预览文件后点「⬇️ 下载」
- **文件夹**：可以建文件夹分类管理文件

## ⚙️ 后台原理（知道的越多越安心）

### 消息流程

```
妈妈发消息
    ↓
存入 Supabase 数据库
    ↓
Cron 每 15 秒检查一次
    ↓
找到消息 → 【阶段A】秒回确认「收到啦！」
    ↓
查对话历史 → AI 生成完整回复
    ↓
【阶段C】发送完整回复
    ↓
标记已处理
```

### 预期耗时
- **确认消息**：~15 秒内弹出
- **完整回复**：15~30 秒（视内容长度而定）
- **超时兜底**：180 秒没回复会显示「阿夏睡着了，让楠楠叫阿夏起床」

## 📝 写小说技巧

虾虾擅长：
- ✍️ 写开篇、结尾、过渡章节
- 🔄 续写（接着上一段写）
- 🔍 扩写（把一段写得更详细）
- 🎭 写对话、打斗、风景描写
- 💡 帮想情节、构思人物

直接告诉虾虾想写什么就行，比如：
> 「虾虾帮我写一个修仙小说开头，主角是废柴逆袭」
> 「帮我把这一段扩写得更有画面感」
> 「我想写一个师徒对决的场景」
